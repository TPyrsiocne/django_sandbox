words_I_know

intended functionalty:

words_I_know will keep track of users and what words each user knows.
each user can add words they know and supply a deffinition. these words will then be available to all users.
a user can see a list of all words they know
a user can see a list of all words with the ones they know marked/indicated somehow

====internals====
there will be a global words table with all the words in the system.
